# 🍕 Pizza Tracker - React Frontend

Este es el cliente frontend para el sistema de gestión de órdenes **Pizza Tracker**. Está diseñado para consumir una API REST construida en Go, utilizando **Server-Sent Events (SSE)** para ofrecer una experiencia en tiempo real sin recargar la página.

## 🚀 Stack Tecnológico

- **Framework:** [React 18](https://react.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/) (Rendimiento ultrarrápido)
- **Estilos:** [Tailwind CSS v4](https://tailwindcss.com/) (Motor nativo Vite)
- **Enrutamiento:** React Router DOM (Próximamente)

## ⚡ Características Principales (En desarrollo)

- Conexión en tiempo real con el backend mediante `EventSource`.
- Interfaz responsiva y moderna utilizando diseño utilitario.
- Consumo de API REST protegida por CORS.

## ⚙️ Instalación y Uso

1. Clonar el repositorio:
   ```bash
   git clone https://github.com/diazjohan98/pizza-tracker-frontend](https://github.com/diazjohan98/pizza-tracker-frontend)
   cd pizza-tracker-front-react
   ```
2. Instalar dependencias:

```
Bash
npm install
```

3. Ejecutar el servidor de desarrollo:

```
Bash
npm run dev
```
